So! Do you want to know how to get your own blog over AWS S3?

Let's get started!

Phase 1 - Getting and preparing our Blog

I want to remember you this is all about an static Blog, so the first thing we are going to need is that Blog and its content, which we will host on a AWS S3 bucket.

You could have several options to get this job done for our purposes. I will place here what I think are two main:

1. To use the best of your skills and programm this from scratch.

2. To use a template.

Of course, the first one is funny and you can learn a lot, but the second is also funny, and you have some of the work in advance, so you can concentrate in the content, and also to code a little in order to get your blog looking as you want.

So, no more words to say I chose the second one.

I got this excelent (in my oppinion) blog template called Clean Blog totally by free from https://startbootstrap.com/.

You have two options, you can download it or fork it from the Star Boostrap GitHub repo.

Well... I will you a third option: You can clone it or fork from my own forked version of the Star Boostrap repo which contains this pretty same blog content. I was working on the branch udacity-bertelsmann-scholarship-blog-challenge

So, I will show you next how I cloned locally my forked repo and started to work on it:

$ git clone git@github.com:javarb/startbootstrap-clean-blog.git

$ git checkout -b udacity-bertelsmann-scholarship-blog-challenge Edit the blog site in your favorite editor. I advise Visual Studio Code To edit all content you also can fork my forked repo and work by your own ove it ;) Go to unsplash and choose images do you like (remember to give credits!)
